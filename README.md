# 💊 Medical Appointment System (FastAPI)

## 📌 Project Overview
This project is a backend system built using FastAPI to manage doctors, schedule appointments, and track consultations.

It demonstrates real-world API development with proper structure, validation, and workflows.

---

## 🚀 Features Implemented

### ✅ Day 1 — GET APIs
- Home route
- Get all doctors
- Get doctor by ID
- Doctors summary
- Get all appointments

### ✅ Day 2 — POST + Pydantic
- Appointment creation using Pydantic models
- Field validation (min_length, gt, defaults)

### ✅ Day 3 — Helper Functions
- find_doctor()
- calculate_fee()
- filter_doctors_logic()
- find_appointment()

### ✅ Day 4 — CRUD Operations
- Create appointment
- Update appointment
- Delete appointment

### ✅ Day 5 — Multi-step Workflow
- Book appointment
- Confirm appointment
- Complete appointment

### ✅ Day 6 — Advanced APIs
- Search (doctors & appointments)
- Sorting (fee, name, experience)
- Pagination (page, limit, total_pages)
- Combined browsing (search + sort + pagination)
